make dir
sudo mkdir -p /var/www/thinktechware.com/html

change owner of dir 
sudo chown -R $USER:$USER /var/www/thinktechware.com/html

transfer website cotent to dir 

create server block files , copy existing to new
sudo cp /etc/nginx/sites-available/saharasoft.co.ke /etc/nginx/sites-available/thinktechware.com

open new created file to edit 
sudo nano /etc/nginx/sites-available/thinktechware.com

create shortcut linking  by typing:

sudo ln -s /etc/nginx/sites-available/thinktechware.com /etc/nginx/sites-enabled/

Next, test to make sure that there are no syntax errors in any of your Nginx files:

sudo nginx -t
If no problems were found, restart Nginx to enable your changes:

sudo systemctl restart nginx
